# Supervised Learning: End-to-End Demo

This repository contains a simple end-to-end supervised learning demo focused on predicting customer churn.

## Goal

The goal is to show a clean, practical machine learning workflow for a binary classification problem from raw data to a saved model.

## Files

- `Supervised Learning - End-to-End Demo.ipynb`: main notebook with the full workflow
- `churn_model_v1.pkl`: exported trained model

## What the Notebook Covers

- business problem definition
- exploratory data analysis
- train/validation/test split
- data cleaning and feature engineering
- baseline model creation
- model training and comparison
- hyperparameter tuning
- final evaluation on the test set
- model export with pickle

## Dataset

The notebook uses the Telco Customer Churn dataset.

Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn/data

## How to Use

1. Open the notebook in Jupyter or VS Code.
2. Run the cells from top to bottom.
3. Install any missing Python packages if prompted, including `xgboost`.
